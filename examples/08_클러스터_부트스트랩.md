# 08. 클러스터 부트스트랩

## 목적

스펙에 정의된 PVC를 네임스페이스에 생성합니다.

## 파이프라인 기준

```bash
kfp-workflow cluster bootstrap \
  --spec configs/pipelines/mambasl_cmapss_smoke.yaml
```

## 벤치마크 기준

```bash
kfp-workflow cluster bootstrap \
  --type benchmark \
  --spec configs/benchmarks/mambasl_cmapss_kepler_smoke.yaml
```

## 튜닝 기준

```bash
kfp-workflow cluster bootstrap \
  --type tune \
  --spec configs/tuning/mambasl_cmapss_tune.yaml
```

## 드라이런

```bash
kfp-workflow cluster bootstrap \
  --type tune \
  --spec configs/tuning/mambasl_cmapss_tune.yaml \
  --dry-run
```

## 동작 방식

- 스펙에서 `storage.*`와 `runtime.namespace`를 읽음
- 필요한 PVC manifest를 출력
- `--dry-run`이 아니면 namespace를 만들고 PVC를 apply
