# 04. 파이프라인 실행 모니터링

## 기본 명령

```bash
kfp-workflow pipeline run list
kfp-workflow pipeline run get <run-id-or-prefix>
kfp-workflow pipeline run wait <run-id-or-prefix>
kfp-workflow pipeline run logs <run-id-or-prefix>
kfp-workflow pipeline run terminate <run-id-or-prefix>
```

## 실험 목록

```bash
kfp-workflow pipeline experiment list
```

## 자주 쓰는 흐름

제출 직후 상태 확인:

```bash
kfp-workflow pipeline run get <run-id>
```

완료까지 대기:

```bash
kfp-workflow pipeline run wait <run-id>
```

특정 스텝 로그 확인:

```bash
kfp-workflow pipeline run logs <run-id> --step train
```

## 참고

- `pipeline run` 명령은 학습과 벤치마크 실행 모두를 모니터링할 때 사용됩니다.
- 일부 명령은 전체 ID뿐 아니라 고유 prefix도 받을 수 있습니다.
