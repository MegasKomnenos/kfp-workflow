# Katib Manifests

This directory is the package-local landing area for manifests generated by `timemixer-new katib render`.

Render a reviewable manifest:

```bash
timemixer-new katib render \
  --spec configs/experiments/fd_all_core_default.yaml \
  --dataset FD001 \
  --output kubeflow/katib/fd001_default_katib.yaml
```

Submit directly from the package CLI:

```bash
timemixer-new katib submit \
  --spec configs/experiments/fd_all_core_default.yaml \
  --dataset FD001
```

Preview the generated manifest without applying it:

```bash
timemixer-new katib submit \
  --spec configs/experiments/fd_all_core_default.yaml \
  --dataset FD001 \
  --dry-run
```

This package is not currently wired into the root `kfp-workflow` plugin registry, so these manifests are standalone package artifacts rather than root-integrated workflow entrypoints.
