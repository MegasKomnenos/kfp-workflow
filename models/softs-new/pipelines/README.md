# Compiled KFP Pipelines — SOFTS New

This directory holds compiled KFP v2 pipeline YAML files generated by:

```bash
softs-new pipeline compile \
  --spec configs/experiments/fd001_smoke.yaml \
  --output pipelines/fd001_smoke.yaml
```

Compiled files are **not** committed to the repository (`.gitignore`).
Regenerate them locally or in CI before submitting to the cluster.

## Available experiment specs

| Spec file | Description |
|-----------|-------------|
| `configs/experiments/fd001_smoke.yaml` | Single-dataset smoke test (2 epochs, CPU) |
| `configs/experiments/fd_all_core_default.yaml` | Full FD001-FD004 default HPO workflow |
| `configs/experiments/fd_all_core_aggressive.yaml` | Full FD001-FD004 aggressive HPO workflow |

## Submitting a pipeline

```bash
# Compile then submit
softs-new pipeline compile \
  --spec configs/experiments/fd_all_core_default.yaml \
  --output pipelines/fd_all_default.yaml

kfp run create \
  --experiment-name softs-cmapss \
  --pipeline-package-path pipelines/fd_all_default.yaml
```
