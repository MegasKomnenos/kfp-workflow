# Compiled Pipelines

This directory is the package-local landing area for YAML generated by `softs-new pipeline compile`.

Example:

```bash
softs-new pipeline compile \
  --spec configs/experiments/fd001_smoke.yaml \
  --output pipelines/fd001_smoke.yaml
```

The package itself does not expose a standalone `pipeline submit` command. For integrated cluster submission, use the root `kfp-workflow` project instead.
